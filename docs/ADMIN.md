# Документация по админке

Папка: `admin/`

## Страницы

- `login.php` — вход.
- `index.php` — очередь SMS/USSD и настройки шлюза (только admin).
- `numbers.php` — номера доступа (просмотр/редактирование/CSV).
- `logs.php` — просмотр логов (очистка только admin).
- `users.php` — управление пользователями и ролями (только admin).
- `firmware.php` — сервисный раздел прошивки (только admin).
- `logout.php` — выход.

## Роли

- `admin`:
  - полный доступ ко всем разделам;
  - управление пользователями, токенами, настройками, очисткой логов.
- `user`:
  - `numbers.php`: просмотр/фильтр/редактирование/импорт/экспорт CSV;
  - `logs.php`: только просмотр;
  - без доступа к `users`, `firmware`, `index` (редирект/ограничение).

## Безопасность

- Сессии:
  - `httponly`, `samesite=Lax`, `secure` при HTTPS;
  - `session_regenerate_id(true)` после логина.
- CSRF:
  - единый токен в сессии;
  - проверка всех POST в `include.php`;
  - hidden token во всех POST-формах.
- Базовые security headers:
  - `X-Frame-Options: SAMEORIGIN`
  - `X-Content-Type-Options: nosniff`
  - `Referrer-Policy: same-origin`
- Анти-брутфорс входа:
  - счётчик в сессии;
  - временная блокировка после серии ошибок.

## Пользователи

Таблица `users`:
- `id`
- `username`
- `password_hash`
- `role` (`admin` | `user`)
- `created_at`

При старте админки для старых БД есть авто-добавление колонки `role`.

## Работа с номерами (`numbers.php`)

- Inline-редактирование строк таблицы.
- Связка номера с адресом и номером участка (таблица `number_locations`).
- Фильтры: ID, поиск номера по фрагменту, группа, активность, адрес, участок.
- Массовые операции: смена группы / удаление по фильтру.
- CSV:
  - единый формат с экспортом:
    - `id;phone_number;group_id;active;address;plot_number;created_at;updated_at`
  - при импорте с `id`:
    - update по id, если найден;
    - insert с этим id, если не найден;
  - без `id` — upsert по `phone_number`.

## Стили и адаптивность

- Единый файл `admin/style_admin.css`.
- Общая тема всех страниц, включая login.
- Адаптация под ширины:
  - `<=1024px`, `<=768px`, `<=480px`.
- На мобильных таблицы уходят в горизонтальный скролл, формы перестраиваются в одну колонку.
