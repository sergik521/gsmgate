# Gate Flash Project

Документация проекта вынесена в папку `docs/`:

- `docs/SKETCH.md` — работа Arduino-скетча, синхронизация, логика звонков/команд.
- `docs/ADMIN.md` — админка, роли, безопасность, работа с номерами.
- `docs/API.md` — API endpoints, авторизация, форматы запросов/ответов.

Дополнительно:
- `.env.example` — шаблон переменных окружения для подключения к БД.
